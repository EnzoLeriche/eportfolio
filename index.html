1er projet

ANALYSE DES RÉSULTATS DES JEUX OLYMPIQUES - LIENS ENTRE LA RÉUSSITE ET DES INDICATEURS SOCIO-ÉCONOMIQUES

En équipe de 5 personnes, nous avons utilisé une base de données couvrant les Jeux olympiques de 1924 à 2010, que nous avons analysée à l'aide du langage de programmation R. Notre objectif principal était d'évaluer si les indicateurs socio-économiques d'un pays pouvaient potentiellement influencer le nombre de médailles remportées par les pays participants. En outre, nous devons identifier les pays qui ont excellés aux Jeux olympiques et comparer les performances entre les éditions d'été et d'hiver pour dégager des tendances.

J'ai joué un rôle actif dans la création du code R utilisé pour notre analyse. Ce code comprenait des analyses descriptives, principalement axées sur les tests de corrélation, et comprenait la création de graphiques pour visualiser clairement les relations entre les différentes variables et les tendances observées.

Pour présenter nos résultats, nous avons préparé une présentation PowerPoint que nous avons partagée avec un professeur, accompagnée d'un rapport écrit détaillant notre méthodologie, nos découvertes et nos résultats.
